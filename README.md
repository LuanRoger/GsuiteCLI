# WorkOffice
Create new Docs, Sheets, Slides and Forms on your terminal.

# Installation
You can install via winget:
```powershell
winget install workoffice
```
Or manually in [releases](https://github.com/LuanRoger/WorkOffice/releases).

# How to use
```powershell
workoffice <TOOL>
```
